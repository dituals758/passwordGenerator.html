# 🔐 Генератор паролей

**Версия 20260220 • Релиз 20.02.2026**

Бесплатное PWA‑приложение для создания надёжных паролей прямо в браузере. Работает полностью офлайн, все данные остаются на вашем устройстве.

## ✨ Возможности

- **🔐 Безопасная генерация** — пароли создаются криптографически случайно на вашем устройстве
- **📱 Полностью офлайн** — работает без интернет‑соединения
- **⚡ Мгновенное копирование** — пароль сразу копируется в буфер обмена
- **🎨 Автоматические темы** — тёмная и светлая тема, синхронизируется с системой
- **🔄 Установка как PWA** — работает как нативное приложение на всех устройствах
- **📱 Адаптивный дизайн** — корректное отображение на iOS, Android, Windows, macOS, Linux
- **➕ Дополнительные опции** — исключение похожих символов (0O, 1lI|) и запрет повторов подряд
- **💾 Сохранение настроек** — все параметры запоминаются между сессиями
- **↻ Сброс настроек** — кнопка быстрого возврата к значениям по умолчанию в едином стиле
- **🚀 Быстрый отклик** — оптимизированная анимация и отзывчивость на тач‑устройствах
- **🔔 Уведомления в стиле iOS** — тосты с анимацией вылета из верхней части экрана
- **🎯 Современный футер** — компактное размещение информации, иконки GitHub, темы и установки
- **🎨 Единый дизайн элементов** — ползунок длины пароля оформлен как остальные переключатели; каждый переключатель на отдельной строке для удобства
- **⚙️ Улучшенный алгоритм генерации** — гарантированное наличие всех выбранных типов символов

1. Откройте приложение в браузере (Chrome, Edge, Safari)

## ⚙️ Как использовать

1. Выберите длину пароля (8–32 символа)
2. Выберите типы символов: строчные/заглавные буквы, цифры, специальные символы
3. При необходимости включите дополнительные опции:
   - *Исключить похожие символы* — убирает визуально похожие символы (0, O, 1, l, I, |)
   - *Исключить повторение символов подряд* — запрещает одинаковые символы рядом
4. Нажмите «Создать и скопировать» — пароль появится в поле и скопируется в буфер обмена
5. Используйте скопированный пароль где нужно
6. При необходимости установите приложение для быстрого доступа через иконку в футере
