# 🔐 Генератор паролей

**Версия 20260502 • Релиз 02.05.2026**

Бесплатное прогрессивное веб-приложение (PWA) для создания надёжных паролей прямо в браузере. Работает полностью офлайн, все данные остаются на вашем устройстве. Теперь с эмодзи‑иконками вместо SVG — идеальное отображение на всех устройствах.

## ✨ Возможности

- **🔐 Безопасная генерация** — пароли создаются криптографически случайно на вашем устройстве
- **📱 Полностью офлайн** — работает без интернет‑соединения
- **⚡ Мгновенное копирование** — пароль сразу копируется в буфер обмена
- **🎨 Автоматические темы** — тёмная и светлая тема, синхронизируется с системой, без мерцания при загрузке
- **🔄 Установка как PWA** — работает как нативное приложение на всех устройствах
- **📱 Адаптивный дизайн** — корректное отображение на iOS, Android, Windows, macOS, Linux
- **➕ Дополнительные опции** — исключение похожих символов, запрет повторов подряд, режим «только уникальные символы»
- **💾 Сохранение настроек** — все параметры запоминаются между сессиями
- **↻ Сброс настроек** — кнопка быстрого возврата к значениям по умолчанию
- **🔔 Системные уведомления** — информация о действиях выводится через встроенные уведомления браузера (при наличии разрешения)
- **🚀 Быстрый отклик** — оптимизированная анимация и отзывчивость на сенсорных устройствах
- **🎯 Минималистичный футер** — компактное размещение информации, иконки GitHub, темы и установки
- **🎨 Дизайн iOS 26** — обновлённые иконки, матовое стекло, адаптация под Dynamic Island и safe areas
- **📦 Упрощённый интерфейс** — плавные уведомления, единый стиль элементов

## 🚀 Быстрый старт

### Онлайн версия
Откройте в браузере: [https://dituals758.github.io/passwordGenerator.html](https://dituals758.github.io/passwordGenerator.html)

### Установка как приложение
1. Откройте приложение в браузере (Chrome, Edge, Safari)
2. Нажмите кнопку установки в футере или используйте меню браузера
3. Приложение появится на рабочем столе/главном экране

## 🛠️ Технические особенности

- **PWA‑ready** — Service Worker, Web App Manifest, автономная работа
- **Современный код** — ES2023+, CSS Custom Properties, семантическая верстка
- **Доступность** — ARIA атрибуты, клавиатурная навигация, фокус-индикаторы
- **Кроссплатформенность** — оптимизировано для всех ОС и браузеров
- **Безопасность** — данные не передаются в сеть, криптографическая генерация
- **Дизайн iOS 26** — матовое стекло, обновлённые векторные иконки заменены на эмодзи для идеальной кросс‑платформенности
- **Сохранение состояния** — настройки хранятся в localStorage

## ⚙️ Как использовать

1. Выберите длину пароля (8–32 символа)
2. Выберите типы символов: строчные/заглавные буквы, цифры, специальные символы
3. При необходимости включите дополнительные опции:
   - *Исключить похожие символы* — убирает визуально похожие символы (0, O, 1, l, I, |)
   - *Исключить повторение символов подряд* — запрещает одинаковые символы рядом
   - *Использовать только уникальные символы* — все символы в пароле будут разными
4. Нажмите «Создать и скопировать» — пароль появится в поле и скопируется в буфер обмена
5. Используйте скопированный пароль где нужно
6. При необходимости установите приложение для быстрого доступа через иконку в футере
